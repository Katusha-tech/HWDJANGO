Обоснование выбора индексов

Поле Service

1. Service.price - ускорит запросы на фильтрацию и сортировку услуг по цене, 
если идет поиск услуг в определенном ценовом диапазоне;

2. Service.is_popular - ускорит выборку популярных услуг для отображения на главной странице или в разделе "Популярные услуги".

Поле Master

1. Master.is_active - ускорит фильтрацию доступных мастеров при отображение списка;

2. Master.experience - ускорит сортировку и фильтрацию мастеров по опыту работы, если идет поиск опытных специалистов.

Поле Order
1. Order.client_name - ускорит поиск заказов по имени клиента, используется в форме поиска на странице orders_list.html;
2. Order.phone - ускорит поиск заказов по телефону клиента, используется в форме поиска на странице orders_list.html;
3. Order.status - ускорит выборку заказов по статусу, используется в форме поиска на странице orders_list.html;
4. Order.date_created - улучшит производительность при сортировке заказов по дате создания (уже используется в ordering);
5. Order: status + appointment_date
   - ускорит запросы, которые фильтруют заказы по статусу и дате записи одновременно;
   - используется для поиска предстоящих записей с определенным статусом, например, "подтвержденные записи на следующую неделю";
   - полезен для административной панели, где менеджеры могут просматривать заказы, 
     отфильтрованные по статусу и отсортированные по дате записи;
6. Order: client_name + phone
   - ускорит запросы, которые ищут заказы по имени клиента и телефону одновременно;
   - используется в административной панели для быстрого поиска всех заказов конкретного клиента;
   - полезен, когда менеджер выбирает несколько чекбоксов в форме поиска на странице orders_list.html.

Поле Review

1. Review.rating - ускорит фильтрацию и сортировку отзывов по рейтингу, что важно для отображения лучших отзывов;
2. Review.is_published - ускорит фильтрацию опубликованных отзывов, что необходимо для отображения только одобренных отзывов на сайте;
3. Review: master + rating
   - ускорит запросы, которые фильтруют отзывы по конкретному мастеру и сортируют их по рейтингу;
   - полезен для аналитики, когда нужно быстро получить средний рейтинг мастера или найти лучшие/худшие отзывы о конкретном мастере.
